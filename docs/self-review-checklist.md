# PR Self-Review Checklist

## Title & Scope
- [ ] Title is imperative and under 72 characters
- [ ] PR contains one logical change only

## Description
- [ ] Clear What + Why + How + Test
- [ ] Any design decisions or tradeoffs explained

## Testing
- [ ] Steps to test provided
- [ ] pytest passes locally

## Code Quality
- [ ] No debug prints
- [ ] Clean naming and indentation
- [ ] No dead/commented-out code
